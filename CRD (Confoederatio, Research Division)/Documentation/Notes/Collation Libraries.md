Due to the amount of data that needs to be processed, Collation should use LODs outside of core mil/financial/news aggregation tasks.

Google Maps (Traffic):
- https://dime-worldbank.github.io/googletraffic/
- Google Maps can be scraped for traffic. Note that here it provides 'Typical traffic' schedules for areas. This should ideally only be done upon user query. Additionally, traffic layers can be toggled on/off, and the different pixels in the image used for point cloud detection from which traffic can be binned into squares.
- Mapbox provides similar traffic services, but its use is more unclear.
