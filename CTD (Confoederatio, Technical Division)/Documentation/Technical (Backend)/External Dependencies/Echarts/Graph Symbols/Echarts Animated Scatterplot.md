Known in Echarts as 'effectScatter'.

- .ripple_effect: Object - Echarts: `.options.series[n].rippleEffect`
	- .brush_type="fill": string - Echarts: `options.series[n].rippleEffect.brushType` - Either 'fill'/'stroke'.
	- .colour: string - Echarts: `options.series[n].rippleEffect.color`
	- .number=3: number - Echarts: `options.series[n].rippleEffect.number`
	- .period=4: number - Echarts: `options.series[n].rippleEffect.period`
	- .scale=2.5: number - Echarts: `options.series[n].rippleEffect.scale`
