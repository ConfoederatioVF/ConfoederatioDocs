Often referred to as `lineStyle` internally by Echarts.

- .stroke_colour=...: string - Echarts: `color/borderColor`
- .stroke_opacity=1: number - Echarts: `opacity` - Value from 0 to 1.
- .stroke_pattern="solid": string - Echarts: `type/borderType` - Either 'solid'/'dashed'/'dotted'.
- .stroke_width=2: number - Echarts: `width/borderWidth`
  
- .stroke_cap="butt": string - Echarts: `cap/borderCap` - Either 'butt'/'round'/'square'.
- .stroke_dash_offset: number - Echarts: `dashOffset/borderDashOffset`
- .stroke_join="bevel": string - Echarts: `join/borderJoin` - Either 'bevel'/'round'/'miter'.
- .stroke_miter_limit=10: number - Echarts: `miterLimit/borderMiterLimit`

- .shadow_blur: number - Echarts: `shadowBlur`
- .shadow_offset_x: number - Echarts: `shadowOffsetX`
- .shadow_offset_y: number - Echarts: `shadowOffsetY`
