Often referred to as `areaStyle` internally by Echarts.

- .fill_colour="#000": string - Echarts: `color`
- .origin="auto": number|string - Echarts `origin` - Either 'auto'/'start'/'end'. Determines where the start of the fill begins in horizontal terms.
- .opacity=0.7: number - Echarts `opacity` - Value from 0 to 1.

- .shadow_blur: number - Echarts: `shadowBlur`
- .shadow_offset_x: number - Echarts: `shadowOffsetX`
- .shadow_offset_y: number - Echarts: `shadowOffsetY`
